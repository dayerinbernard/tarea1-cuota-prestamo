Que hace el programa.

Este programa calcula la cuota mensual de un prestamo utilizando el monto del prestamo, la tasa de interes anual y el tiempo de pago en meses. El resultado muestra el monto del prestamo, la tasa de interes, el tiempo de pago y la cuota mensual que debe pagar el cliente
